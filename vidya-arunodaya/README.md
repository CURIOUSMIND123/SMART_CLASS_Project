# 🌄 Vidya Arunodaya — Offline Smart-Class Library (demo)

**What you install on the smart board. Runs 100% offline from board storage or a pendrive.**

Open `index.html` in any modern browser (Chrome/Edge) — no internet, no install, no server.
On a real Android board it runs the same way inside a WebView/kiosk launcher.

## What it is
A curriculum launcher: **Class → Subject → NCERT chapter**. Each chapter opens real,
market-grade interactive content — all bundled legally with attribution:

- **🔬 Simulations** — real **PhET** interactive sims (University of Colorado), in **English AND Hindi**.
  These are the same class of assets Digi-Kaksha itself preloads. CC BY 4.0, offline, attributed.
- **🧊 3D models** — a fully-offline 3D viewer (drag/rotate/zoom, animated) proving real 3D models
  play on the board. Production plugs in licensed/CC educational models.
- **✨ Original lessons** — our own interactive 3D lessons (The Human Eye, PlantVerse) — the standard
  we build state-owned original content to.
- **📖 Read** — original chapter summaries + a link that opens the official NCERT chapter when online
  (never copied or repackaged).
- **✅ Quiz** — interactive MCQs; in a deployed board the score is recorded for the district dashboard.
- **Languages** — English / हिंदी / **Nyishi** (local-language sample) toggle in the top bar.

## How to demo (2 minutes)
1. Open `index.html`. Point at "Works 100% offline" and the language toggle.
2. Class 10 → Science → **Electricity** → the PhET Circuit Kit loads — build a live circuit on the board.
3. Switch the top bar to **हिंदी**, reopen a sim — same simulation, Hindi.
4. Class 10 → Science → **The Human Eye** → our original interactive 3D lesson.
5. **3D Model Lab** → rotate the animated 3D model. Then **Read** tab → show the "Open official NCERT" link.
6. Any chapter → **Quiz** → answer a question.
